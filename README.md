## üèõ Project Philosophy
**Vista Suites** is more than a landing page; it's a digital translation of high-end physical hospitality. The objective was to take a sophisticated brand identity (delivered via PDF) and engineer a "living" digital environment. 

As a **Full-Stack Developer**, I approached this project with a "Platform-First" mindset: ensuring that the front-end isn't just a skin, but a high-performance interaction layer optimized for conversion and brand authority.

## üõ† Engineering Stack
*   **Architecture**: Optimized Vanilla Engine (Zero Dependency).
*   **Visual Logic**: Modern CSS3 with CSS Variables for theme consistency.
*   **Typography**: Boutique font-face integration (`TAN-Mon-Cheri` & `Red Hat Display`).
*   **Asset Management**: Lazy-loaded high-resolution architectural photography.
*   **Icons**: Scalable Vector Graphics (SVG) via Lucide.js for crispness on Retina displays.

## üé• Motion Design Principles (The "WOW" Factor)
The "Premium" feel is achieved through intentional, orchestrated motion. I implemented a custom interaction engine that handles:
*   **Intelligent Reveal System**: Custom `.reveal-mask` and `.reveal-stagger` classes that utilize the **Intersection Observer API** for high-performance scroll triggers.
*   **Interactive Cursor System**: A dual-layered custom cursor with variable inertia, creating an organic connection between the user and the interface.
*   **Cinematic Preloading**: A bespoke preloading sequence that manages asset state, ensuring the viewport never exhibits "Flash of Unstyled Content" (FOUC).
*   **Depth Engineering**: Multi-layered parallax on the hero typography to create a 3D architectural sense.
